## 🎯 **About Goodlink Germany**

**Good-Link Germany GmbH** is the premier European gateway for high-quality medical devices and automotive components sourced directly from trusted Chinese manufacturers. Located in the heart of Europe in Cologne, Germany, we are the first Good-Link branch outside of China, founded in 2020 through strategic cooperation with German shareholders and management.

### 🌟 **Our Mission**
To become the best channel for European companies seeking reliable partnerships with Chinese suppliers, specializing in:
- 🏥 **Premium Medical Devices** (MDR/CE certified)
- 🚗 **Automotive Components** (EMC/ROHS compliant)
- ⚡ **Electronic Components** (High-precision manufacturing)
- 🔧 **Industrial Solutions** (Complete supply chain management)

## 🚀 **Getting Started**

### 📋 **Prerequisites**
- Node.js 20+ 
- npm or yarn
- Git

### 💻 **Installation**
```bash
# Clone the repository
git clone https://github.com/Mlaiel/goodlink-germany.git
cd goodlink-germany

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### 🌐 **Available Scripts**
```bash
npm run dev      # Development server
npm run build    # Production build
npm run preview  # Preview build
npm run lint     # Code linting
npm run deploy   # Deploy to GitHub Pages
```
